# Дата: 2025-02-25

- **Что было сделано:**
  Моя текущая задача была добавить общий app layout для protected страниц как dashboard, library, profile и т.д.
  Она должна была включать sidebar с меню, переключателями, аватаром и именем пользователя и обертку для основного содержимого страницы.
  В мобильной версии sidebar должен открываться с помощью burger-menu.
  При этом я поправила router.ts чтобы страница имела этот layout если она protected (роут имеет этот флаг).
  Также я перенесла инициализацию theme из header в app.ts так как это оказалось правильнее всего, чтоб отрабатывало для всех роутеров.
- **Проблемы:**
  Прямо проблем не было. Но надо было подумать как реализовать этот app layout чтобы он применялся к определённым страницам. Оказалось есть несколько вариантов.
  Через rout как я и реализовала (с флагом). Также можно было наследоваться от layout, но тогда надо было наследовать каждую станицу отдельно.
- **Решения (или попытки):**
  Я выбрала такой способ, так как он предполагает что сам layout не будет перерисовываться.
  Иконки сгенерила с помощью AI, самый быстрый вариант и легче сделать в необходимом стиле.
