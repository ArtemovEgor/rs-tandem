# Первые коммиты
Добавил функцию для адаптива scss. Добавил дефолтную структуру для бэкенда. Внес изменения в конфиг-файл ESLint.

По итогам встречи обсудили дальнейшие планы. Первая серьезная фича, за которую берусь в рамках полноценного бэкенда - авторизация и аутентификация пользователя. Для фронта должна быть предусмотрена страница с личными данными. Пока не понятно, нужно ли делать настоящую регистрацию, с подтверждением по почте. Наверное первым этапом стоит сделать упрощенную регистрацию, примерно как было в таске "Fun Chat", а дальше, если получится, сделать уже настоящую. Коллеги начинают делать базовые интерфейсы в том числе для блока авторизациеи/аутентификации. Главное не забыть использовать их, а не выдумать собственные.

По планам в течении текущей начавшейся недели разобраться с базовой авторизацией/аутентификацией, чтобы уже можно было использовать совместно с фронтенд-частью.
